Description:
Abstract:
This ARC/INFO polygon coverage contains the second level administrative boundaries at a 1:1000000 scale.

The coverage is part of the Famine Early Warning System (FEWS) / Associates in Rural Development (ARD) African series.

Kenya is associated with the Horn region of Africa.

Purpose:
This data set is to be used in conjunction with other layers of Kenya data. The data are intended to show the spatial distribution of the second level administrative boundaries for Kenya.

The data are to be used in modeling, targeting and demonstration projects. The data are also used in natural resource management, inventory, change analysis, policy making, environmental awareness and land use planning.

Supplemental information:
Procedures_Used: The 1979 Census Enumeration Area maps, with sub-location boundaries and the Survey of Kenya Topo. Map Series (1:50000), were used as the base maps for creating new 1989 Census Enumeration Maps with sub-location boundaries. A large field mapping effort, headed by the Kenya Bureau of Statistics (Cartographic Section), began in 1987 in preparation for the 1989 Census. Census enumeration agents visited all 3667 sub-locations and met with the village Assistant Chiefs for the purpose of updating/ correcting the 1979 sub-location boundaries.

The corrected sub-location boundaries were then plotted (hand-drawn) onto the 1:50000 Survey of Kenya, Topo. Map Series. Direct tracing and photographic procedures were later used to transfer the sub-location line work to a series of district maps of varying scales. District map scales were as follows:

Urban Areas 1:50000 (i.e. Mombassa & Nairobi) High Population Density Rural Areas 1:100000 Medium Population Density Areas and Arid and Semi-Arid Areas 1:250000

The sub-location boundaries on each district map were then digitized. In general, registration errors during digitizing (RMS) did not exceed .003 inches. Each sub-location polygon was given a code reflecting its Admin. context (i.e., Province, District, Division, Location, sub-location). This coding allowed for the subsequent creation of higher order admin. maps; i.e., linework was "dissolved" to create the Province borders.

These country specific sub-national boundaries were then integrated into the DCW master country database using the ARC/INFO command UPDATE.

The country specific data set, containing all of the sub-national hierarchy, was extracted from the DCW master data set using the ARC/INFO command RESELECT and the second level administrative units were extracted from the country specific data set using the ARC/INFO command DISSOLVE.

A master data set, based on the Digital Chart of the World (DCW) PONET layer, was created. Burkina Faso, Mali, Ethiopia, and Eritrea are included in the master data set, however, these countries have international boundaries that have been revised.

Overview description:
Entity and attribute overview:
The item (ID) defines the unique administrative unit code, as defined by FEWS/ARD. The item (CNT-ABB) defines the unique country abbreviation, as defined by EDC. The item (ADMN2) defines the second level administrative unit names, as defined by FEWS/ARD. The item (NUM-CODE) defines the unique numeric administrative unit identifier, as defined by EDC. The item (REGION) defines the African region that Kenya is associated with, as defined by FEWS/ARD. The item (FEWS-ABB) defines the unique country abbreviation, as defined by FEWS/ARD. The item (FEWSID) defines the unique administrative unit identifier, as defined by FEWS/ARD. The item (FEWSID2) defines the unique identifiers for the second level administrative units, as defined by FEWS/ARD.
